# Pokemon API

## Dev deploy

1. Clone repository
2. Execute

```
npm run install
```

3. Have Nest CLI installed

```
npm i -g @nestjs/cli
```

4. Start database - check `docker-compose.yml` for database information

```
npm run docker:up
```

## Stack implemented for this proyect

- MongoDB
- Nest
